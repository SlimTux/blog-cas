 ## How to write an html button?

> ```html
> <button class="button-save large">Big Fat Button</button>
> ```

<button class="button-save large">Big Fat Button</button>

 ## How to center the html button?
* Just use the "center" tag
> ```html
> <center></center>
> ```
### Results

> ```html
> <center><button class="button-save large">Big Fat Button</button></center>
> ```
